# React + TypeScript + Vite + TailwindCSS

## informartion
node version: >=6.9.0

React version: 18.2.0

## Getting Started

### Installation 

install repo
```bash
git clone https://github.com/Hsan25/FrontendDevReactjs-Hasan.git
```

open repo
```bash
cd FrontendDevReactjs-Hasan
```

install

```bash
npm install
# or
yarn install
# or
pnpm install
# or
bun install
```
run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
